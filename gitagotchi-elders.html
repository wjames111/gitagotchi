gitagotchi · the elder beard — one rule, fifty chins

Life stage elder (account ≥ 8 years) grows a beard. It's not drawn per species: the renderer finds the face (the KK mouth below the eyes, else the PP beak, else two rows under the eyes), then hangs a tapered age-gray beard from that anchor with a mustache at the corners. Fixed gray — never the linguist color — so age reads the same on every pet.

Anchor algorithm (same in bash): eye row = first row containing a KW/WK pair · mouth = first KK run below the eye row (ignores dark caps above, e.g. the ladybug) · else the beak's PP run · else eye row + 2, centered between the eyes. Mustache pixels flank the anchor; the beard tapers 8→6→4 wide below it (grand-elder adds two forked rows). Beard paints over the body — beards hang. Glasses: a gray bridge across the eye row plus one frame pixel outside each eye.
