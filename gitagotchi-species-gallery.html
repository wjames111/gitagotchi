gitagotchi · pixel species gallery

Fifty species on the same 24×18 palette-indexed grid as the dense-mode axolotl — half-block rendering in the terminal (24 cols × 9 rows each). Built as body archetypes × feature modules (ears, horns, tails, wings, shells), per the design doc's sprite plan. Every sprite is two idle frames + a derived blink; letters mean the same thing in every file (O body · D outline · W light · K eyes/dark · P accent · R blush · S shade), so patterns, accessories and faces compose identically across all of them. Every species has the full state set from the design doc — switch states below (or press 1–6) and the whole zoo follows: sleep, eat, sick, celebrate, and the shared hibernation cocoon are derived transforms of each species' base frames, exactly how the bash renderer would compute them.

Same grid, your language — the linguist recolor

The body palette is generated from the linguist color of the user's top language (outline = −35% lightness, light = +60%, shade = −12%), so one grid serves every user. This is Kilu's blob grid wearing six different top languages.
